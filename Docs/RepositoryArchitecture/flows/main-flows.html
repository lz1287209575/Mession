主链路 1：登录并进入世界

这是最能代表系统总装配关系的一条链路。

Client Login FlowC++-style Runtime Call Chain
Client
  -> Gateway.Client_Login
      -> World.WorldClient::Client_Login
          -> StartDeferredFiber(...)
              -> DoClientLogin(...)
                  -> Login.IssueSession(...)
                  -> PlayerService.PlayerEnterWorld(...)
                      -> DispatchRuntimeCommand(...)
                          -> DoPlayerEnterWorld(...)
                              -> Login.ValidateSession(...)
                              -> Mgo.LoadPlayer(...)
                              -> Scene.EnterScene(...)
                              -> Router.UpsertPlayerRoute(...)
                              -> QueueScenePlayerEnterNotify(...)
  <- ClientLoginResponse

主链路 2：切场

World 负责编排，Scene 负责执行，Router 负责投影。

Switch Scene FlowC++-style Runtime Call Chain
Client
  -> Gateway.Client_SwitchScene
      -> World.WorldClientCommon
          -> StartDeferredFiber(...)
              -> PlayerService.PlayerSwitchScene(...)
                  -> DispatchRuntimeCommand(...)
                      -> DoPlayerSwitchScene(...)
                          -> Scene.LeaveScene(old)
                          -> Scene.EnterScene(target)
                          -> Router.UpsertPlayerRoute(target)
                          -> QueueScenePlayerLeave / Enter / Update Notify(...)
  <- ClientSwitchSceneResponse

主链路 3：战斗施法

World 负责跨玩家顺序和结果提交，Scene 负责战斗执行。

Combat FlowC++-style Runtime Call Chain
Client
  -> Gateway.Client_CastSkill
      -> World.WorldClientCombat
          -> StartDeferredFiber(...)
              -> PlayerService.CastSkill(...)
                  -> DispatchRuntimeCommandMany(caster, target)
                      -> DoCastSkill(...)
                          -> EnsureCombatAvatar(caster)
                          -> EnsureCombatAvatar(target)
                          -> Scene.CastSkill(...)
                          -> CommitCombatResult(...)
  <- ClientCastSkillResponse

主链路 4：登出

登出已经不是简单的离线，而是一个状态收敛过程。

Logout FlowC++-style Runtime Call Chain
Client
  -> Gateway.Client_Logout
      -> World.WorldClientCommon
          -> StartDeferredFiber(...)
              -> PlayerService.PlayerLogout(...)
                  -> BuildLogoutParticipants(...)
                  -> DispatchRuntimeCommandMany(...)
                      -> DoPlayerLogout(...)
                          -> CleanupPlayerSocialState(...)
                          -> Scene.LeaveScene(...)
                          -> Mgo.SavePlayer(...)
                          -> PlayerCommandRuntime.BumpEpoch(...)
                          -> RemovePlayer(...)
  <- ClientLogoutResponse
